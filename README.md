# Manikya Pant — Portfolio

Personal portfolio website. Built with HTML, CSS, and Vanilla JavaScript.

**Live site:** https://manikya-pant.github.io

---

## Structure

```
portfolio/
├── index.html       ← Main page
├── css/style.css    ← All styles
├── js/main.js       ← All JavaScript
└── assets/          ← Add resume.pdf here
```

## Hosting on GitHub Pages

1. Create a repo named exactly: `manikya-pant.github.io`
2. Push all files to the `main` branch
3. Go to repo **Settings → Pages → Source → main branch**
4. Your site is live at `https://manikya-pant.github.io`

## To update
Edit `index.html` directly — swap in your real GitHub/LinkedIn URLs,
update project links, and add more projects as you build them.
